==============================================
  CLÍNICA EQUILÍBRIO MENTAL — SITE PRONTO
==============================================

ARQUIVOS:
  index.html  → Arquivo principal do site (tudo em um único arquivo)

COMO PUBLICAR:
  1. Faça upload do arquivo index.html para a raiz do seu servidor
     (geralmente a pasta "public_html" ou "www")
  2. Acesse seu domínio no navegador e o site já estará no ar

HOSPEDAGENS SUGERIDAS:
  - Hostinger   → hostinger.com.br
  - Hostgator   → hostgator.com.br
  - Netlify      → netlify.com (gratuito)
  - Vercel       → vercel.com (gratuito)

ANTES DE PUBLICAR — substitua no index.html:
  ┌─────────────────────────────────────────────────────────┐
  │ (11) 99999-9999     → Número real do WhatsApp           │
  │ 5511999999999       → DDD + número sem espaços          │
  │ https://instagram.com/ → Link real do Instagram         │
  │ @equilibriomental   → Seu @ do Instagram                │
  │ A definir           → Endereço real da clínica          │
  └─────────────────────────────────────────────────────────┘

  Dica: Use Ctrl+H no Notepad++ ou VS Code para substituir.
